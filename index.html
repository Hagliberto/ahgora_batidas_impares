✦ GUIA RÁPIDO

Guia rápido das
Batidas Ímpares

Importe os resultados do TOTVS Ahgora, localize as pendências com filtros, consulte os detalhes e gere arquivos organizados para conferência. Todo o processamento acontece localmente no navegador.

1
Acesse o Analytics

No TOTVS Ahgora, siga o caminho PontoWeb › Analytics › Painéis e abra o card “Batidas Ímpares”.

2
Baixe os resultados

Na tabela “Lista de Batidas Ímpares”, use os três pontos, escolha JSON, CSV ou XLSX e mantenha os dados formatados.

3
Importe o arquivo

Use “Importar arquivo”. Depois do carregamento, o mesmo botão passa a funcionar como “Trocar arquivo”.

4
Refine a consulta

Abra o expander de análise para usar cards clicáveis, filtros, ranking por colaborador e distribuição por status.

5
Confira as ocorrências

Alterne entre Tabela e Cards. O botão de detalhes aparece antes da coluna Dia e mostra o registro completo.

6
Exporte com finalidade

Gere PDF, PNG ou XLSX detalhado. As exportações respeitam o recorte ativo e preservam a diferenciação visual dos status.

Fluxo recomendado

  • Baixe o arquivo com “Manter os dados formatados” marcado.
  • Confira os cards e aplique os filtros necessários.
  • Abra os detalhes das ocorrências antes de exportar.
  • Use o XLSX para conferência e o PDF/PNG para apresentação.

Privacidade e boas práticas

  • Nenhum dado é enviado para servidores pela aplicação.
  • Utilize o painel em equipamento confiável.
  • Valide os filtros antes de gerar os arquivos finais.
  • Ao concluir, limpe os dados quando o computador for compartilhado.
Os arquivos permanecem no navegador. A aplicação funciona localmente e não transmite os registros importados.
