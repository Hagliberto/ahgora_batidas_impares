# Segurança e Privacidade

## Processamento local

Os dados importados são processados no navegador. A aplicação não contém chamadas de rede para envio de informações.

## Persistência

A última importação pode ser armazenada em `localStorage`. Isso significa que os dados permanecem no perfil local do navegador até serem removidos pela aplicação ou pelas configurações do navegador.

## Recomendações

- utilize computadores autorizados;
- evite importar dados reais em computadores compartilhados;
- use **Limpar dados** ao encerrar o trabalho em ambientes compartilhados;
- mantenha o navegador atualizado;
- revise os arquivos antes de compartilhá-los.

## Arquivos exportados

PDF, PNG e XLSX podem conter dados pessoais e funcionais. A proteção, armazenamento e compartilhamento desses arquivos são responsabilidade do operador.
